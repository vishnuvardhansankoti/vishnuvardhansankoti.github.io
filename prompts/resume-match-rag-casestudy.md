While Gen AI and cloud llm are very powerful tools that we can utilize for any task, it is not always cost effective to do so. also, there is chance of PII data leaving on prem systems which in some industries like fiannce and insurance is a very serious issue. 

I was working in my current startup to use our Azure Credits to create an AI tool for our recruiting team to match a candidates profile against a given job description to get a fair understanding how the candidate matches against the job description. During intial POC phases, we tried some cloud llm agents for sample resumes against sample job descriptions, but very soon we figured out that it is both not cost effective and also a risk of getting data going out of our systems. 

Then we approached mini local llm model and cosine similarity match
